# 🎬 Movie-CRUD-App

A simple backend application to perform CRUD operations on movies using **Express.js** and **MongoDB**.
  
## 📦 Features
- Add a new movie 🎥   
- Get all movies 📄
- Update movie info ✏️
- Delete a movie ❌ 

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
- Dotenv
